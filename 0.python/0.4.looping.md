# Looping

Execute n lines of code until the condition turn to False

```python
num = 1


while num <= 10:
  print(num)
  num += 1
```

For each element of a list do something
```python
for i in range(1, 11):
  print(i)
```

* range is a function that return a list from the first arg to 2nd (the 2nd not included).
